Server-auth only (no client cert) per the design's in-tunnel-auth decision; the client verifies the server cert against a configured CA.

```go
// Package gosttls provides GOST TLS (TLS 1.2 + GOST ciphersuites) via a CGO
// binding to OpenSSL 3.x and the "gost" engine. It exposes net.Conn-compatible
// client and server connections that plug into the transport Dialer.
//
// Requires OpenSSL 3.x dev libraries and the gost engine
// (Debian: libssl-dev + libengine-gost-openssl).
package gosttls

/*
#cgo pkg-config: libssl libcrypto
#cgo CFLAGS: -Wno-deprecated-declarations
#include <openssl/ssl.h>
#include <openssl/err.h>
#include <openssl/engine.h>
#include <stdlib.h>

static ENGINE *gvpn_load_gost_engine(void) {
    ENGINE *e = ENGINE_by_id("gost");
    if (e == NULL) return NULL;
    if (ENGINE_init(e) == 0) {
        ENGINE_free(e);
        return NULL;
    }
    // Make GOST digests/ciphers/pkeys available by default.
    ENGINE_set_default(e, ENGINE_METHOD_ALL);
    return e;
}
*/
import "C"
import (
	"fmt"
	"sync"
)

var (
	initOnce   sync.Once
	initErr    error
	gostEngine *C.ENGINE
)

// Init performs one-time OpenSSL + gost engine initialization. It is safe to
// call repeatedly and concurrently; only the first call does work. All gosttls
// constructors call it implicitly.
func Init() error {
	initOnce.Do(func() {
		if C.OPENSSL_init_ssl(0, nil) != 1 {
			initErr = fmt.Errorf("gosttls: OPENSSL_init_ssl failed: %s", lastError())
			return
		}
		e := C.gvpn_load_gost_engine()
		if e == nil {
			initErr = fmt.Errorf("gosttls: failed to load gost engine (install libengine-gost-openssl): %s", lastError())
			return
		}
		gostEngine = e
	})
	return initErr
}

// Version returns the linked OpenSSL version string.
func Version() string {
	return C.GoString(C.OpenSSL_version(C.OPENSSL_VERSION))
}

// lastError drains the OpenSSL error queue into a string for diagnostics.
func lastError() string {
	code := C.ERR_get_error()
	if code == 0 {
		return "no OpenSSL error"
	}
	buf := make([]byte, 256)
	C.ERR_error_string_n(code, (*C.char)(C.CBytes(buf)), 256)
	// Re-fetch using a C buffer to avoid CBytes leak ambiguity.
	cbuf := (*C.char)(C.malloc(256))
	defer C.free(unsafe_Pointer(cbuf))
	C.ERR_error_string_n(code, cbuf, 256)
	return C.GoString(cbuf)
}
```

## Task 3: Conn + Dial + Listener (the core CGO)

**Files:** Create `core/gosttls/conn.go`.

Design:
- `type Conn struct { ssl *C.SSL; file *os.File; raw net.Conn; closeOnce sync.Once }` implementing `net.Conn`.
- Connection model (blocking): dial/accept the TCP socket in Go, obtain a blocking dup fd via `raw.(*net.TCPConn).File()`, `SSL_set_fd(ssl, C.int(file.Fd()))`, then `SSL_connect`/`SSL_accept`. Keep `raw` to manage lifetime; do I/O through `SSL_*`.
- `Read(p)`: `n = SSL_read(ssl, &p[0], len(p))`; on `n<=0` map `SSL_get_error` to `io.EOF` (clean close) or a wrapped error.
- `Write(p)`: loop `SSL_write` until all bytes sent or error.
- `Close()` (once): `SSL_shutdown`, `SSL_free`, `file.Close()`, `raw.Close()`.
- `LocalAddr/RemoteAddr/SetDeadline/SetReadDeadline/SetWriteDeadline`: delegate to `raw` (deadlines on the underlying socket bound blocking SSL ops).
- `Dial(ctx, network, addr string, cfg Config) (*Conn, error)`: `Init()`; net.Dial; newClientCtx; `SSL_new`; if `cfg.ServerName != ""` set SNI (`SSL_set_tlsext_host_name`) and enable hostname verification (`SSL_set1_host`); set fd; `SSL_connect`; on error free everything.
- `type Listener struct { net.Listener; ctx *C.SSL_CTX }`; `Listen(network, addr string, cfg Config) (*Listener, error)`; `Accept()` wraps the accepted conn: `SSL_new`, set fd, `SSL_accept`.
- `CipherName(*Conn) string` helper (`SSL_get_current_cipher` → `SSL_CIPHER_get_name`) for tests to assert a GOST cipher.

Acceptance: compiles; exercised by Task 4's integration test.
